# Come creo una richiesta di pagamento a rate?

{% hint style="info" %}
**ID articolo:** M-014 · **Area:** Esercente · **Categoria:** Crea un pagamento a rate
{% endhint %}

## Passaggi

1. Dalla pagina Pagamenti, seleziona "Crea una richiesta di pagamento".
2. Inserisci l'importo e scegli "Paga a rate" come tipologia.
3. Seleziona il numero di rate: vedrai in automatico l'importo per rata e le commissioni applicate.
4. Scegli il canale — POS o Telefono — se hai un Qomodo POS attivo.
5. Inserisci i dati del cliente (nome, cognome, codice fiscale, email, telefono, indirizzo): devono essere completi e corrispondere a un'identità italiana.
6. Invia: il cliente riceve un link via SMS o WhatsApp per confermare il piano.

## In sintesi

Le commissioni sulla rateizzazione sono sempre a carico dell'esercente: il cliente non le vede né le paga mai, indipendentemente dal canale scelto.
