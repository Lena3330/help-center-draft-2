# Qual è la differenza tra Paga ora via POS e via Telefono?

{% hint style="info" %}
**ID articolo:** M-033 · **Area:** Esercente · **Categoria:** Crea un pagamento immediato
{% endhint %}

## Passaggi

1. Con "Qomodo POS": il cliente paga in negozio, con il tuo supporto. Il POS si attiva automaticamente, senza bisogno di inviare alcun link.
2. Con "Telefono": il cliente paga in autonomia dal proprio smartphone. In questo caso riceverà un link di pagamento via SMS o WhatsApp.
3. Per il canale Telefono ti basta il numero di cellulare del cliente — nessun altro dato è richiesto.

## In sintesi

Scegli POS quando il cliente è fisicamente in negozio; scegli Telefono per vendite a distanza o quando preferisci non gestire il pagamento di persona.
