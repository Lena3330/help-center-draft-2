# Una rata non è stata addebitata: cosa succede?

{% hint style="info" %}
**ID articolo:** C-119 · **Area:** Cliente · **Categoria:** Risolvere un problema
{% endhint %}

## Passaggi

1. Riceverai una notifica che ti informa dell'addebito non riuscito.
2. Controlla che la carta collegata sia valida e abbia fondi sufficienti.
3. Puoi aggiornare i dati della carta seguendo il link ricevuto.
4. Se il problema continua, contatta l'assistenza: nessuna penale viene applicata automaticamente.

## In sintesi

Un singolo addebito non riuscito non blocca il resto del piano: puoi sempre regolarizzare la rata in sospeso.
