# Posso cambiare il numero di rate dopo aver confermato?

{% hint style="info" %}
**ID articolo:** C-136 · **Area:** Cliente · **Categoria:** Confermare un pagamento a rate
{% endhint %}

## Passaggi

1. Una volta confermato il piano, il numero di rate non può essere modificato autonomamente.
2. Se hai bisogno di cambiare il piano, contatta l'assistenza il prima possibile.
3. In alcuni casi è possibile annullare la richiesta prima che venga completata, se ancora in fase di valutazione.

## In sintesi

Per questo è utile controllare bene il piano proposto prima di confermarlo con i dati della carta.
