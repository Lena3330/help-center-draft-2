# Ho ricevuto un link per pagare a rate: cos'è?

{% hint style="info" %}
**ID articolo:** C-101 · **Area:** Cliente · **Categoria:** Confermare un pagamento a rate
{% endhint %}

## Passaggi

1. Il negozio dove hai fatto un acquisto ha creato per te un piano di pagamento a rate.
2. Apri il link ricevuto via SMS o WhatsApp.
3. Controlla l'importo e il numero di rate proposte.
4. Inserisci i tuoi dati e i dati della carta per confermare.
5. Riceverai una risposta in pochi minuti.

## In sintesi

Non devi pagare nessun interesse: l'importo che vedi diviso per le rate è esattamente quello che pagherai, senza costi aggiunti.
